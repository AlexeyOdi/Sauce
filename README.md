Проект посвящен тестированию страницы авторизации сайта https://www.saucedemo.com

1. Чтобы запустить проект, нужно установить Docker Desktop.
2. Затем создать образ Docker. Чтобы это сделать, нужно в консоли зайти в корневой каталог проекта.
3. В консоле ввести команду "docker build -t sause-test .". Дождать когда создатся образ.
4. Затем там же ввести "docker run -p 8080:8080 sause-test". Дождаться окончание тестов.
5. Чтобы посмотреть результаты тестирования, в браузере в адрессную строку вписать "localhost:8080"
6. Чтобы отключить локальный сервер, на котором размещены результаты тестов, в консоле нажать "Ctrl+C"
